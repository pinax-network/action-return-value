# EOSIO Action Return Value

## CDT

- [blanc++ v0.12.0](https://github.com/haderech/blanc)

## Quickstart

```bash
$ blanc++ exxample.cpp
$ cleos set contract <ACCOUNT> . example.wasm example.abi
```

## .env

```env
ACCOUNT="<ACCOUNT>"
PRIVATE_KEY="<EOSIO PRIVATE KEY>"
ENDPOINT="https://jungle4.api.eosnation.io"
```
